## 1. 产品概述

洒水车出没时间预测记录器是一款专为骑车上班族设计的实用工具，帮助用户记录城市洒水车的出没规律，通过数据分析预测各路段洒水车经过时间，让用户提前规划路线避免被溅湿。

### 核心价值
- 解决骑车上班族被洒水车溅湿的痛点
- 通过用户众包数据积累，形成可靠的洒水车时刻表
- 智能分析预测，提前提醒用户避开洒水路段

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地存储 | 记录洒水车出没、查看预测时刻表、管理历史记录 |

### 2.2 功能模块
1. **首页仪表盘**：今日预测提醒、快速记录入口、常用路段概览
2. **记录页面**：记录遇到洒水车的时间、路段、是否被溅到
3. **时刻表页面**：按路段展示洒水车预测经过时间
4. **历史记录页面**：查看和管理所有记录的数据
5. **统计分析页面**：展示洒水车出没规律图表、溅水概率等数据

| 页面名称 | 模块名称 | 功能描述 |

## 4. 用户界面设计

### 4.1 设计风格
- **设计主题**：清新水蓝风格，呼应洒水车主题
- **主色调**：水蓝色系（#0EA5E9 主色、#38BDF8 辅助色、#E0F2FE 背景色）
- **强调色**：橙红色（#F97316）用于溅水标记和重要提醒
- **按钮风格**：圆润胶囊型按钮，带微妙阴影和悬停动效
- **字体**：标题使用 Noto Sans SC，正文使用系统字体栈，保持清晰易读
- **布局风格**：卡片式布局，圆角柔和，层次分明
- **图标风格**：使用水滴、时钟、地图等相关图标，统一线性风格

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 首页仪表盘 | 今日预测卡片 | 渐变蓝色背景，时间轴展示，带脉冲提醒动画 |
| 首页仪表盘 | 快速记录按钮 | 悬浮圆形按钮，水波纹点击效果 |
| 首页仪表盘 | 溅水统计 | 卡片式展示，带进度条和百分比 |
| 记录页面 | 表单区域 | 清新的表单控件，时间选择器带水滴动画 |
| 记录页面 | 溅水选择 | 大按钮切换，选中时有明显视觉反馈 |
| 时刻表页面 | 路段卡片 | 卡片列表，每个卡片展示预测时间和概率 |
| 时刻表页面 | 时间轴 | 横向时间轴，不同颜色表示出没概率 |
| 统计分析页面 | 热力图 | 网格热力图，颜色深浅表示出没频率 |
| 统计分析页面 | 趋势图表 | 平滑折线图，带渐变色填充 |

### 4.3 响应式设计
- **移动优先**：采用移动端优先的响应式设计
- **断点设置**：768px（平板）、1024px（桌面）
- **触控优化**：按钮最小高度 44px，间距适中，适合触控操作
- **手势支持**：支持左右滑动切换页面、下拉刷新

### 4.4 动画与交互
- **页面切换**：平滑滑动过渡
- **数据加载**：骨架屏占位，淡入加载
- **按钮交互**：水波纹点击效果、缩放反馈
- **提醒动画**：预测时间临近时脉冲提醒动画
- **图表动效**：数据加载时的渐进式动画
